# Analysis Directory

This directory contains analysis scripts and comparison reports for understanding experiment results and code differences.

## Contents

### Analysis Scripts
- `analyze_all_experiments.py`: Comprehensive experiment results analysis
- `analyze_all_branches.py`: Branch comparison and code analysis

### Comparison Reports
- `MAIN_VS_AVINASH_RESULTS_COMPARISON.md`: Detailed comparison of main branch vs avinash branch results
- `TRUSTWEIGHT_ORGANIZATIONAL_STRUCTURE.md`: Analysis of TrustWeight branch organization
- `OUTSIDE_FOLDER_COMPLETE_ANALYSIS.md`: Complete mapping of outside/ folder experiments
- `FEDBUFF_VS_TRUSTWEIGHT_COMPARISON.md`: Method comparison reports
- `MERGED_BRANCH_IDENTIFICATION.md`: Git branch merge analysis

### Branch Analysis
- `ALL_BRANCHES_COMPLETE_ANALYSIS.md`: Complete analysis of all Git branches
- `BRANCH_CODE_ANALYSIS.md`: Code differences between branches
- `COMPLETE_BRANCH_CODE_ANALYSIS.md`: Comprehensive branch code comparison
- `STALENESS_BRANCH_DETAILED_ANALYSIS.md`: Detailed staleness branch analysis

## Purpose

These analysis tools help:
- Compare experiment results across different branches
- Understand code evolution and differences
- Map experiment locations and results
- Generate comprehensive reports for documentation
