# Experiment Results

This directory contains historical experimental results from hyperparameter tuning and federated learning runs.

## SqueezeNet FedBuff Results

- `squeezenet_fedbuff_results/` - Historical SqueezeNet FedBuff experiments
  - Architecture: SqueezeNet (before migration to ResNet-18)
  - Contains 9 experiment runs with training metrics in CSV format
  - These were used to evaluate FedBuff performance with SqueezeNet architecture

## Hyperparameter Tuning Results

- `resnet18/` - ResNet-18 hyperparameter tuning results
- `*.csv` - SqueezeNet hyperparameter tuning results
